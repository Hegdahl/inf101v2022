\doxysubsection{Detailed Description}
\mbox{\hyperlink{classNim}{Nim}}, a single player game to play against the computer or another humam (easy to make a bot for).

The game consists of several rows of tokens.

In one turn you may remove any positive number of tokens tokens as long as they are from the same row.

The one to remove the last token wins.
